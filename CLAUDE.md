## 문서 업데이트 단축 명령

- `"기획 문서 업데이트"` → `Docs/GameDesign.md` 편집
- `"프로그램 문서 업데이트"` → `Docs/Program.md` 편집
- `"UI 문서 업데이트"` → `Docs/UI.md` 편집
- `"아트 문서 업데이트"` → `Docs/Art.md` 편집
- `"문서 전체 업데이트"` → 위 4개 파일 모두 편집

업데이트 시 사용자에게 **변경할 내용**을 먼저 요약해서 확인받은 후 반영한다.

## 문서 업데이트 리마인드

작업 완료 후 아래 항목이 변경된 경우, 문서 업데이트가 필요한지 사용자에게 물어본다:
- 게임 규칙/흐름/승패 조건 변경 → `GameDesign.md`
- 새 스크립트/인터페이스/데이터 모델/컨벤션 변경 → `Program.md`
- UI 구조/UXML 패널/스와이프 스펙 변경 → `UI.md`
- 컬러/애니메이션/에셋 변경 → `Art.md`
